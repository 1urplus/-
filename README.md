# 🍩 洋洋记账

个人智能记账助手 — 桌面版，支持 Windows 和 macOS。

## ✨ 功能

| 功能 | 说明 |
|------|------|
| 📝 **记一笔** | 记录收入/支出，支持金额、两级分类、日期、备注 |
| 🗣️ **智能输入** | 用自然语言描述，AI 自动解析为记账记录（例："昨天午饭花了30"） |
| 📋 **账单列表** | 按时间倒序展示，支持分类筛选和关键词搜索 |
| 📊 **统计分析** | 月度饼图、每日柱状图、年度趋势图、分类排行榜 |
| 🤖 **AI 分析** | DeepSeek AI 分析消费结构，给出省钱建议 |
| 📁 **数据导出** | 支持导出 CSV 和 Excel 文件 |
| 🏷️ **分类管理** | 内置 15 个一级分类 + 50 个二级分类，支持自定义 |

## 🔒 数据安全

所有数据存储在**你的电脑本地**，不上传到任何服务器。AI 分析仅发送脱敏的汇总数据（不含任何个人身份信息）。

## 🚀 快速开始

### 1. 安装依赖

```bash
npm install
```

### 2. 启动开发模式

```bash
npm run electron:dev
```

### 3. 配置 AI 功能

启动后，在设置页面填入 [DeepSeek API Key](https://platform.deepseek.com) 即可使用智能输入和 AI 分析功能。

### 4. 打包为可执行文件

```bash
# Windows
npm run electron:build:win

# macOS
npm run electron:build:mac
```

打包输出在 `release/` 目录下。
